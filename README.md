# 📌 Overview
This project is built using Databricks Community (Free Edition) to practice data engineering concepts such as data ingestion, transformation, and analysis using Apache Spark.
It demonstrates how to work with distributed data processing using notebooks, Spark APIs, and different file formats.

# 🎯 Objectives
Learn Databricks workspace and notebook environment
Perform data ingestion using multiple file formats
Apply transformations using PySpark
Understand schema handling and data processing
Practice real-world data engineering workflows

# 🛠️ Tech Stack
Platform: Databricks Community Edition
Engine: Apache Spark
Language: Python (PySpark)
Storage: DBFS (Databricks File System)
